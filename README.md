# Piyo Translator (汎用ピヨ語翻訳機)

あらゆるテキストを「ピヨ語」（`ピ`と`ヨ`の羅列）に変換し、完全に元に戻すことができる翻訳機です。
UTF-8バイト列をビット列に変換する可逆変換方式を採用しているため、日本語、英語、絵文字、さらには画像データなど、あらゆる情報をピヨ語化できます。

## 特徴

- **完全可逆変換**: どのようなテキストも情報の欠落なくピヨ語に変換・復元できます。
- **Web UI**: モダンで使いやすいWebインターフェース。
- **CLIツール**: コマンドラインからのパイプ処理などに対応。

## インストール

Python 3.13以上が必要です。
`uv` を使用して依存関係をインストールすることをお勧めします。

```bash
# リポジトリのクローン
git clone https://github.com/NexCairn/piyo-translator.git
cd piyo-translator

# 依存関係のインストール
uv sync
```

## 使い方

### 1. Web UI (ブラウザで利用)

サーバーを起動します。

```bash
uv run uvicorn main:app --reload
```

ブラウザで [http://127.0.0.1:8000](http://127.0.0.1:8000) にアクセスしてください。
テキストを入力して「ピヨ語に変換」ボタンを押すだけで利用できます。

### 2. CLIツール (コマンドラインで利用)

**エンコード (テキスト -> ピヨ語)**
```bash
uv run python -m piyo.cli "こんにちは"
# 出力: ヨヨヨピピピヨヨヨピピピピピピヨヨピピヨピピヨヨヨヨヨピピピヨヨヨピピピピピヨピヨピピヨピピヨヨヨヨヨピピピヨヨヨピピピピピピヨヨピヨピヨピヨヨヨヨヨピピピヨヨヨピピピピピピヨヨピヨピピピピヨヨヨヨピピピヨヨヨピピピピピピヨヨピヨピヨヨヨヨ
```

**デコード (ピヨ語 -> テキスト)**
```bash
uv run python -m piyo.cli -d "ピヨピピヨピピピピヨヨピピヨピヨピヨヨピヨヨピピピヨヨピヨヨピピピヨヨピヨヨヨヨ"
# 出力: Hello
```

**パイプ処理**
```bash
echo "Hello" | uv run python -m piyo.cli
```

## 開発
